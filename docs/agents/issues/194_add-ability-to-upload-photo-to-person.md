# Issue 194: Add Ability to Upload a Photo to a Person

## Description
On the person listing page, the user should be able to click a link that opens a form to upload a photo for that person.

## Expected Behavior
- Each person entry in the listing has a link (e.g., "Upload Photo").
- Clicking the link opens a form with a file input.
- Submitting the form sends a POST request to `/persons/:id/photo.json`.
